# Russian Media Analysis: Sentiment, Propaganda and Media Bias

Dette repoet inneholder et pågående prosjekt der jeg analyserer russisk mediediskurs,
med særlig fokus på krigen i Ukraina.

Mål på sikt:
- Trene en sentimentanalyse-modell for russisk.
- Utforske i hvilken grad propaganda-lignende språkbruk kan identifiseres i tekst.
- Sammenligne statlig kontrollerte medier og mer uavhengige medier.
- Analysere hvilke personer, organisasjoner og steder som nevnes (NER), og i hvilke kontekster.

for å detektere propaganda eller "troll". Etiske og metodiske begrensninger
diskuteres fortløpende i dokumentasjonen.
